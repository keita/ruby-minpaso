ruby-minpaso

Get Version

0.1.0

What

ruby-minpaso is a wrapper library for Minpaso API.

Installing

sudo gem install minpaso

API document

http://minpaso.rubyforge.org/rdoc/

The basics

Search

pager = Minpaso.search(:manufacturer => "Sony")
item = pager.items.first
item.product_name
item.wei_score
item.cpu_score
item.memory_score
item.video_score
item.game_score
item.hdd_score
item.total_score

Details

pcinfo = Minpaso::PCInfo.new(1788)
pcinfo.cpu.score
pcinfo.memory.size.to_mb

How to submit patches

Read the 8 steps for fixing other people’s code and for section 8b: Submit patch to Google Groups, use the Google Group above.
