h1. ruby-minpaso

h2. What

ruby-minpaso is a wrapper library for Minpaso API.

h2. Installing

<pre syntax="ruby">sudo gem install minpaso</pre>

h2. API document

"http://minpaso.rubyforge.org/rdoc/":http://minpaso.rubyforge.org/rdoc/

h2. The basics

h3. Search

<pre syntax="ruby">
pager = Minpaso.search(:manufacturer => "Sony")
item = pager.items.first
item.product_name
item.wei_score
item.cpu_score
item.memory_score
item.video_score
item.game_score
item.hdd_score
item.total_score
</pre>

h3. Details

<pre syntax="ruby">
pcinfo = Minpaso::PCInfo.new(1788)
pcinfo.cpu.score
pcinfo.memory.size.to_mb
</pre>
